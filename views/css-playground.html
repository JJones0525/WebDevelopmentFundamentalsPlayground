CSS Statemnts

Four main components of CSSS statement

  1. Declaration The whole CSS statemtn in curly braces.
  2. Selector:The name of the part you want to style.
  3. Property:properties to edit( e.g., width, height, background,etc.)
  4. Value:numbers, colors, urls - things to give value to property.
